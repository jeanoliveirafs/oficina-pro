# 🔧 Oficina Pro — Auto Repair Shop Management System

> **EN** | [🇧🇷 Versão em Português abaixo](#-sobre-o-projeto-português)

A complete management system for auto repair shops. Oficina Pro streamlines scheduling, service orders, customer management, and inventory — all in one modern web application.

---

## ✨ Features

- 📋 **Service Orders** — Create and manage repair orders with status tracking
- 👥 **Customer Management** — Store customer and vehicle information
- 📅 **Scheduling** — Appointment booking and calendar management
- 📦 **Inventory** — Parts and supplies tracking
- 📊 **Dashboard** — Business overview with key metrics
- 📱 **Responsive Design** — Works on desktop and mobile

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React, TypeScript, Vite |
| UI | TailwindCSS, Shadcn/UI |
| Backend | Supabase (PostgreSQL + Auth) |
| Deploy | Vercel |

---

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/jeanoliveirafs/oficina-pro.git
cd oficina-pro

# Install dependencies
npm install

# Configure environment variables
cp .env.example .env

# Start development server
npm run dev
```

## 🇧🇷 Sobre o Projeto (Português)

**Oficina Pro** é um sistema de gestão completo para oficinas mecânicas. Simplifica agendamentos, ordens de serviço, gerenciamento de clientes e controle de estoque — tudo em uma aplicação web moderna.

### Funcionalidades

- 📋 **Ordens de Serviço** — Crie e gerencie ordens de reparo com rastreamento de status
- 👥 **Gestão de Clientes** — Armazene informações de clientes e veículos
- 📅 **Agendamento** — Reserva de consultas e gestão de calendário
- 📦 **Estoque** — Controle de peças e suprimentos
- 📊 **Dashboard** — Visão geral do negócio com métricas principais

### Como Executar

```bash
git clone https://github.com/jeanoliveirafs/oficina-pro.git
cd oficina-pro && npm install && npm run dev
```
